This program offers a strategic, GUI-based military interface built in Java. This application provides a high-contrast, tactical display for managing mission logs with real-time time tracking, secure data exportation, and a persistent camouflage system.

### ✨ Program Features
*   **Tactical Dark Mode UI:** Designed with a custom "Midnight Black" and "Tactical Green" color palette for a professional military look and feel.
*   **Mission Time Logging:** Integrated real-time date and time stamps that can be appended to the tactical log with a single command.
*   **Secure Data Export:** Built-in file handling that writes the entire session log to a local `log.txt` file for permanent records.
*   **Persistent Camouflage System:** Generates a unique random green hue at launch that is remembered and applied to the entire frame throughout the session.
*   **Tactical Shortcuts:** Full keyboard support using Accelerators (Ctrl+1, Ctrl+2, etc.) and Mnemonics (Alt+O) for rapid command entry.
*   **Exit Protocol:** Integrated confirmation dialogs to prevent accidental loss of tactical data during mission termination.

### 🖥️ Usage Instructions
*   **Log Mission Time:** Select Option 1 from the menu (or press `Ctrl+1`) to print the current mission timestamp into the text box.
*   **Export Data:** Select Option 2 from the menu (or press `Ctrl+2`) to save your current tactical log into the `log.txt` file.
*   **Apply Camouflage:** Select Option 3 from the menu (or press `Ctrl+3`) to apply the session’s unique random green hue to the background and borders.
*   **Terminate Mission:** Select Option 4 from the menu (or press `Ctrl+4`). A **Mission Exit Protocol** prompt will appear; confirm "Yes" to safely shut down the system.
